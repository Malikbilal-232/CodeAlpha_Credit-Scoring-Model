# Credit Scoring Model

## Objective
Predict an individual's creditworthiness using historical financial data.

## Algorithms Used
- Logistic Regression
- Decision Tree
- Random Forest

## Evaluation Metrics
- Precision
- Recall
- F1-Score
- ROC-AUC

## Features
- Income
- Debt
- Payment History

## Tools
Python, Pandas, Scikit-learn, Matplotlib
